COVID DATA ANALYSIS 2021

This CLI application was built using Python. Various Python libraries were used to build this application including BeautifulSoup, sqlalchemy, pandas, etc. MySQL is used with Python to create a local database storing COVID data.
This work was completed as part of my assignment for my Python class. The intention behind this project was to practice using BeautifulSoup to scrape data from a covid statistics website for every country in the world. Pandas is used to compare data between countries using various functions from this data analysis tool.

Code Repository

    The desired experience for a user in the context of this project was to create a CLI application that would compare covid data for a chosen country by a user for various days including yesterday, today, and tomorrow. The compared values for the chosen countries are the deaths toll, the number of cases (difference, %increase, %decrease). Also, the application would provide a comparison of this data with four other countries chosen randomly at runtime. Since this application is CLI-based, the chosen UI is very simple and clean since it consists only of text. All the information is organized to simplify the user reading the output information on the console.
