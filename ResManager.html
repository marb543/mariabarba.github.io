ABOUT THE PROJECT


ResManager is a comprehensive desktop application designed for the rental business sector. Built using Java and a local SQL database, this application streamlines the management of rental properties, tenants, and related business operations.

Financial Management

Track financial institutions and transactions

Contractor Management

Manage and monitor contractor information

Lease Tracking

Organize and monitor tenant leases

Property Database

Maintain detailed property records

DEVELOPMENT PROCESS


The development process followed a structured approach:

  • GUI design using SceneBuilder and FXML
  • Database architecture and SQL implementation
  • Java controllers for each functional section
  • Integration of UI components with backend logic
  • Implementation of data manipulation functions
